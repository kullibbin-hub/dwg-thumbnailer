Скрипт реализует создание превью для файлов формата .dwg для Nautilus. Проверен в Altlinux Workstation 11.1, возможно будет работать в Ubuntu.

Скрипт создает два файла /usr/local/bin/dwg-thumbnail.sh и /usr/share/thumbnailers/dwg.thumbnailer, скачивает NConvert и распаковывает его в /usr/local.

Использование.

По зеленой кнопке "<>Code" вверху страницы выбрать Download zip, скачается архив. Распаковать в домашний каталог (можно оставить в папке "Загрузки"), 
в распакованной папке в свойствах скрипта "dwg-thumbnailer-install.sh" по правому клику включить "Исполняемый как приложение". Правый клик по скрипту - 
копировать, открыть терминал, правый клик в терминал - вставить. Enter. Желательно чтоб скрипт не находился в папке с пробелами, например, 
"Рабочий стол" или "мой софт". Лучше переместить его в корень домашнего каталога, или оставить в папке "Загрузки", как уже писал. 
Также, можно запустить скрипт правым кликом по нему - "запустить как приложение".

После отработки скрипта, для гарантии, выйти из сессии и зайти снова, или перезагрузить компьютер. Теперь nautilus будет показывать превью 
сохраненных проектов в формате .dwg .

Данный скрипт можно свободно распространять, модифицировать и использовать без каких-либо ограничений, на свой страх и риск.
